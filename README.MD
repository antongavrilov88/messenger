Описание
```
Мессенджер. Проектная работа на курсе яндекс-практикум
```
Установка
```
git clone https://github.com/antongavrilov88/mf.messenger.praktikum.yandex.git
```
Тесты
```
npm run test
```
Сборка
```
npm run build
```
Запуск
```
npm run start
```
Сайт
```
По адресу https://agavrilovapp.herokuapp.com актуальная версия.
```
